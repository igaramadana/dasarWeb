# Dasar Pemrograman Web Semester 3

## Deskripsi Mata Kuliah
Mata kuliah "Dasar Pemrograman Web" merupakan pengantar bagi saya untuk memahami konsep dan teknik dasar dalam pengembangan aplikasi web. Dalam mata kuliah ini, saya akan mempelajari berbagai aspek pemrograman web, termasuk HTML, CSS, JavaScript, dan PHP Native.
